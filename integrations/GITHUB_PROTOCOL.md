# GitHub Protocol

This repo is the Figma integration and trial evidence repo for NZ Agency.

## Workflow

1. Figma designs / exports / decisions get documented here.
2. Implementation-ready assets are linked to the relevant company repo.
3. Major usage events are logged in `logs/usage-log.md`.
4. Trial audit on 2026-06-23 reviews this repo for evidence.

## Do not commit

- API keys
- Figma personal access tokens
- OAuth secrets
- Customer PHI
- Private payment or medical data
