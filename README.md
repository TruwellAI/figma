## Trial protocol

Every tool we add follows the same standard:

1. Record monthly cost and billing card.
2. Track actual usage.
3. Track projects/assets produced.
4. Measure whether it improved speed, quality, or automation.
5. Report back after 30 days with evidence.
6. Keep only if useful.

## What Figma must prove

- Better website/app design quality
- Faster approval workflows
- Useful design-to-code handoff
- Reusable NZ Agency design systems
- Practical GitHub/OpenClaw integration
