# NZ Agency Tool Trial Protocol

Every new creative/build tool must prove it earns its subscription.

## Rule

Tools are month-to-month trials unless Neil explicitly makes them permanent.

For each tool we track:

1. Monthly cost
2. Billing card
3. Trial start date
4. Review date, usually +30 days
5. What the tool is supposed to improve
6. Actual usage
7. Outputs created
8. Whether it improved quality, speed, automation, or client approvals
9. Keep / reallocate / upgrade / cancel recommendation
