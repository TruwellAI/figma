## Core stack recommendation based on integration proof

### Buy/connect first
1. **Figma** — source of truth + MCP/API. Already trialing.
2. **Builder.io / Visual Copilot** — best bridge from design to production code/CMS.
3. **FAL API** — best API-first creative model hub.
4. **ElevenLabs** — API-first voice layer.
5. **Creatify** — API-first UGC video ads.
6. **v0** — strong GitHub/Vercel code workflow; verify API/automation before making it permanent.

### Useful, but prove manually before paying long-term
- Framer
- Bolt/StackBlitz
- Spline
- Rive
- Runway
- Relume

### Do not prioritize until API/MCP/export automation is proven
- Uizard
- Visily
- Arcads
- Google Stitch
